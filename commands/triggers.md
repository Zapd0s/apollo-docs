# Triggers

| Command                       |                                              |
| ----------------------------- | -------------------------------------------- |
| `a!triggers add <trigger>`    | Adds a Trigger.                              |
| `a!triggers remove <trigger>` | Removes a Trigger.                           |
| `a!triggers`                  | Shows The Available Triggers For The Server. |
