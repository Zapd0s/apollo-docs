# Giveaways

| Command                             | Information                             |
| ----------------------------------- | --------------------------------------- |
| `a!gstart <time> <winners> <prize>` | Starts a Giveaway.                      |
| `a!greroll` <`message_id>`          | Rerolls a Winner Of a Certain Giveaway. |
| `a!gend <message_id>`               | Ends a Giveaway.                        |
